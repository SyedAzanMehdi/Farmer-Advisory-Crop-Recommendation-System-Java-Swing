# Farmer-Advisory-Crop-Recommendation-System-Java-Swing
A professional, enterprise-style Java Swing-based Farmer Advisory System designed to provide agricultural guidance and crop recommendations. The project focuses on clean architecture, user-friendly UI, and real-world problem solving, specifically targeting farmers in Mianwali District, Pakistan.
# Project Objectives

Provide structured agricultural advisory using a desktop application

Apply MVC architecture in a real-world Java project

Design a professional UI/UX using Java Swing & AWT

Strengthen understanding of event handling, data structures, and software design

# Tech Stack

Language: Java

GUI: Java Swing, AWT

Architecture: MVC (Model–View–Controller)

IDE: NetBeans / IntelliJ IDEA / Eclipse

Data Handling: In-memory data structures (Lists, Maps)

# Key Features

Secure Login System

Role-based access (Admin/User)

Crop recommendation and advisory module

Clean and professional GUI design

Fully implemented ActionListeners

Modular and scalable code structure

User-friendly navigation and layouts
